Employee Tracker Dashboard

Ek powerful React-based dashboard jo company ke employees ka data manage karne ke liye banaya gaya hai. Isme filter aur search jaise features hain jo real-time mein kaam karte hain.

Key Features:
* **Dynamic Search:** Naam ya role ke basis par employees ko turant filter karein.
* **Salary Tracking:** Har employee ki salary aur designation ka clear view.
* **Component-Based UI:** React components ka use karke clean aur modular code.
* **Responsive Design:** Bootstrap use kiya gaya hai taaki laptop aur tablet dono par sahi dikhe.

Tech Stack:
* **React.js:** Frontend library logic ke liye.
* **JavaScript (ES6+):** Array methods (filter, map) ka use.
* **Bootstrap:** Styling aur responsive layout ke liye.
* **CSS3:** Custom styling aur table formatting.
